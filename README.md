# English

### Preface:

A progress bar is a visual resource widely used in computing. It passes information on the percentage of progress of a particular process. The idea behind this library is to deploy an easy-to-use C ++ progress bar using the object-oriented feature.

# Português

### Introdução

Uma barra de progresso é um recurso visual amplamente utilizado na computação. Ela passa a informação do percentual de progresso de um determinado processo. A idéia dessa biblioteca é implantar uma barra de progresso em C++ de fácil utilização, utilizando para tal o recurso de orientação a objetos.
